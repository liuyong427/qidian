公司简介

成都启点科技有限公司，由从事通信研发近十年的资深技术人员组成，专业从事新一代IP通信产品和企业融合通信系统研发、生产和销售，致力于为政府和企业提供先进、智能的融合通信一体化解决方案。

我们兼收并蓄通讯技术及企业管理实践经验，立足于自主知识产权产品，是为了满足企事业用户对通讯业务日益多样化、高集成化的需求。目前已开发的产品有：数字程控交换机IPPBX、呼叫中心系统、电话营销CRM、电话会议系统、网络电子传真群发系统、语音催费查询系统等。

目前已成功研发具有自主知识产权的“启点-zswitch”系列通信产品：数字程控交换机IPPBX、呼叫中心系统、电话营销CRM、电话会议系统、网络电子传真群发系统、语音催费查询系统等，特别是针对保险行业开发的呼叫中心系统、电力催费系统在实际应用中取得了良好的社会效益。
